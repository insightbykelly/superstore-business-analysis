# Superstore Business Analysis 📊

## 📌 Project Overview
This project analyzes the Superstore dataset to identify key drivers of profitability, evaluate advertising opportunities, and understand product return behavior. The goal is to provide actionable insights to improve business performance and reduce losses.

---

## 🎯 Objectives
- Identify major profit centers and loss-makers  
- Recommend products and subcategories to retain or discontinue  
- Evaluate advertising opportunities based on profitability  
- Analyze return patterns to identify high-risk products and customers  

---

## 🛠️ Tools Used
- Tableau  
- Google Sheets  

---

## 📊 Analysis Breakdown

### 🔹 Profit & Loss Analysis
- Identified top-performing and underperforming product combinations  
- Highlighted products contributing to consistent losses  
- Recommended subcategories to focus on and eliminate  

### 🔹 Advertising Analysis
- Determined the most profitable states and months for advertising  
- Analyzed average profit trends over time  
- Estimated optimal advertising spend based on return on ad spend (ROAS)  

### 🔹 Returns Analysis
- Created a return rate metric using calculated fields  
- Identified products and customers with the highest return rates  
- Analyzed the relationship between profit and return rate across different dimensions

## 🚀 Key Insights
- Certain products and subcategories consistently generate losses and should be reconsidered  
- Profitability varies significantly by region and time, impacting advertising effectiveness  
- High return rates are concentrated among specific products and customers  
- Some high-return items negatively impact overall profitability  

---

## ✅ Conclusion
This analysis provides data-driven recommendations to help the superstore reduce losses, optimize product offerings, and make informed decisions about advertising and operations.
